📚 Digital Library - Book Data Generator

Convert your Google Drive books into JavaScript code

📋 How to use:

  1. Open your Google Drive folder in a new tab
  2. For each author:
    • Enter the author name
    • For each book, right-click the PDF → Share → Copy link
    • The link looks like: https://drive.google.com/file/d/FILE_ID_HERE/view
    • Extract the FILE_ID and enter it below
  3. Click Generate Code
  4. Copy the generated code into your website
💡 Tip: To quickly get FILE_ID: Copy the share link, look between /d/ and /view
