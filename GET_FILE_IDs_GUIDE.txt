STEP 1: OPEN YOUR GOOGLE DRIVE FOLDER
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Go to: https://drive.google.com/drive/folders/1wow9tOtg7eOvmLEKUxHNwmBUTvl4Nk1V?usp=drive_link


STEP 2: FOR EACH PDF FILE, DO THIS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  1. Right-click on the PDF file
  2. Select "Get link"
  3. A dialog opens showing the link (default: "Viewer" is selected)
  4. Copy the link
  5. Open it in a new tab (or look at the URL)


STEP 3: EXTRACT THE FILE ID
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

The URL will look like this:
https://drive.google.com/file/d/[FILE_ID_HERE]/view?usp=sharing

EXAMPLE:
https://drive.google.com/file/d/1qnWbvn-TmExBwEJvjqKXzJcKJKqKGgVl/view?usp=sharing
                               ├─────────────────────────────────┤
                                      This is your FILE ID


STEP 4: UPDATE data.js
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Replace "PASTE_FILE_ID_HERE" with your actual file ID

BEFORE:
{ title: "Hamlet", fileId: "PASTE_FILE_ID_HERE", size: "2.5 MB" }

AFTER:
{ title: "Hamlet", fileId: "1qnWbvn-TmExBwEJvjqKXzJcKJKqKGgVl", size: "2.5 MB" }


STEP 5: SAVE & TEST
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

✓ Save the file (Ctrl+S)
✓ Refresh the website (F5)
✓ Click on a book and try "Read Online" or "Download"
✓ It should work!

💡 TIPS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

• File IDs are long strings of numbers and letters
• They usually start with "1" or "10"
• The ID stays the same even if you rename the file
• If you get an error, double-check the ID (no spaces!)
